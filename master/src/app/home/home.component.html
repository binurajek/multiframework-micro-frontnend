Multi-Framework Micro Frontend Architecture

This dashboard demonstrates a robust Micro Frontend (MFE) implementation using Module Federation. It seamlessly integrates independent applications built with different UI frameworks into a single "Shell" environment.

Angular MFE

A native Angular micro-frontend. It shares core dependencies with the Shell to minimize bundle size while maintaining isolation.

React MFE

A React application integrated into this Angular Shell safely using a generic MfeWrapperComponent to mount its independent root.

How It Works

1. Module Federation

Webpack's MF Plugin allows sharing and loading remote entry files at runtime dynamically.

2. MFE Wrapper Strategy

A custom element safely bootstraps external libraries (ReactDOM.createRoot) internally without leaks.

3. Shared Dependencies

Libraries like RxJS/Angular are defined as singletons to stop duplicate downloads between shell and guests.

4. Global Error Handling

Fault-tolerant middlewares display "Retry" components if any single remote fails resolving.
